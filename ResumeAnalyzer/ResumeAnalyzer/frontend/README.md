## Overview

Dynamic CV Analyzer for HRs is a web-based application designed to help recruiters automate resume parsing, job description management, and candidate–job matching.  
It extracts key information from resumes, manages job postings, and calculates match scores based on skill relevance.

---

## Modules Completed

### Module 1: Resume Parsing & Extraction
- Implemented secure resume upload (PDF/DOCX).  
- Extracts and displays candidate details (name, contact, education, skills, experience, projects).  
- Generates a structured skill matrix for HR review.  

### Module 2: Job Description & Management
- Allows HRs to create, edit, and manage job descriptions.  
- Stores and retrieves JDs through a dashboard interface.  
- Includes a skill-based resume–job matching system with percentage scoring and feedback options.
